## 1. 準拠方針

ZeroTrust-ID-Governance システムは、情報セキュリティマネジメントシステム（ISMS）の国際規格である **ISO/IEC 27001:2022** への準拠を目指す。

本システムが取り扱うアイデンティティ情報・アクセス権限・認証データは、組織の情報セキュリティにとって最重要資産であり、以下の管理策を実装・維持する。

### 準拠スコープ

| 項目 | 内容 |
|------|------|
| 対象システム | ZeroTrust-ID-Governance（バックエンド / フロントエンド / DB） |
| 対象データ | ユーザー情報・アクセス権限・認証情報・監査ログ |
| 対象環境 | Azure AKS（本番）/ GitHub Actions（CI/CD） |
| 除外範囲 | エンドユーザー端末 / 外部 IdP（Entra ID / AD）の内部管理 |
